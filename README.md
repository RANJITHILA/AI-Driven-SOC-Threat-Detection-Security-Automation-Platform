# AI-Driven SOC Threat Detection & Security Automation Platform

## Overview
An end-to-end AI-powered SOC simulation platform built using Python, Machine Learning, Flask, and Docker.

## Features
- Security log generation
- Threat preprocessing pipeline
- AI anomaly detection
- Real-time alert generation
- SOAR automated response
- Threat intelligence enrichment
- SOC web dashboard
- Docker deployment

## Tech Stack
Python, Pandas, Scikit-learn, Flask, Docker, WSL Ubuntu

## Use Cases
- SOC Analyst learning
- Threat Hunting simulation
- Security Automation practice
- AI in Cybersecurity portfolio
